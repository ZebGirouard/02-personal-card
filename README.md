# 02 Personal Card

## Goal
Turn plain HTML into a polished learner profile card and practice the HTML/CSS loop: edit, refresh, inspect, adjust.

## Start Here
1. Open `index.html` in your browser.
2. Replace the placeholder text, stats, image, and links.
3. Make the layout work on small screens by finishing the missing mobile styles in `styles.css`.

No install step is required.

## Stretch Goals
- Add a favorite projects section.
- Add hover styles to the links.
- Change one type or spacing choice to improve hierarchy.
