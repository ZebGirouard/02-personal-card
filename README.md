# 02 Personal Card

## Goal
Turn plain HTML into a simple personal card and practice the HTML/CSS loop: edit, refresh, inspect, adjust.

## Start Here
1. Open `index.html` in your browser.
2. Replace the placeholder text, image, and links.
3. Make the card look better on small screens by finishing the mobile styles in `styles.css`.

No install step is required.

## Stretch Goals
- Add a favorite projects section.
- Change the color palette.
- Add hover styles to the links.
